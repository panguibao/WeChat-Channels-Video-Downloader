软件简介：
微信视频号下载工具，支持下载视频号视频、直播回放、直播，支持获取视频下载链接、自动监听微信视频号视频。

核心功能：
全面覆盖：支持下载普通视频号视频、直播回放和实时直播
多种下载方式：支持直接下载或获取下载链接自行下载
智能监听：自动识别微信中打开的视频号内容
标题显示：可选开启视频标题显示功能

使用教程：
1. 开始监听视频号视频
点击软件中的”开始监听”按钮，然后在微信中打开视频号，软件会自动识别并捕获视频信息。（如果需要下载直播，点击开始监听按钮就行）

2. 下载视频
方式一：直接在软件中点击”下载”按钮，或勾选”自动下载视频”选项
方式二：点击”复制链接”，使用第三方下载工具保存后，再通过本工具解密视频

3. 开启标题显示（可选）
1.点击”开启标题显示”按钮
2.打开缓存文件夹，安装两个证书.p12 和 证书-cert.p12（双击证书，一路点确定）
如果证书未生成，需手动打开mitmdump.exe后再试

3.重启软件，看到”开启成功”提示后即可显示视频标题
